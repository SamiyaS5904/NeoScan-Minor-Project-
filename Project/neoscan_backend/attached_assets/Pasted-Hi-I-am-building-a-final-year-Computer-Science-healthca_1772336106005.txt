PROJECT OVERVIEW (Core Workflow)

NeoScan works in 3 stages:

Calibration Step

User must first capture an image of a white surface (e.g., white wall or paper).

This image is sent to backend for lighting normalization calibration.

Backend stores calibration reference.

Frontend must clearly guide the user through this step.

Baby Image Capture Step

User captures image of baby’s eye region (or forehead).

Camera preview must work properly using browser MediaDevices API.

Image preview shown before submission.

Image is sent to backend for processing.

Analysis & Result

After submission, user is redirected to Dashboard.

Show "Analyzing..." loading state (simulate 1–2 minute processing delay).

Poll backend periodically until result is ready.

Display:

Bilirubin value (mg/dL)

Risk Level (Safe / Moderate / High)

Confidence percentage

Risk meter visualization (circular or gradient scale)

Graph of recent trend
